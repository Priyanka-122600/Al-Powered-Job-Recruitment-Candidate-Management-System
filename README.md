# AI-Powered Job Recruitment & Candidate Management System

## 📌 Project Overview
The AI-Powered Job Recruitment & Candidate Management System is a Salesforce-based application designed to streamline and automate the recruitment process. The system enables recruiters and hiring managers to efficiently manage job openings, candidate applications, interview workflows, offer approvals, and hiring decisions using Salesforce Admin features and Agentforce AI.

## 🎯 Objectives
- Centralize recruitment data in Salesforce.
- Automate hiring workflows using Flows.
- Improve candidate evaluation using AI Candidate Score.
- Implement approval processes for high salary offers.
- Enhance recruiter productivity with email alerts and automation.
- Provide secure role-based access to recruitment records.

## 🚀 Workflow
1. Create Job Opening
2. Register Candidate
3. Submit Candidate Application
4. AI evaluates candidate
5. Recruiter reviews application
6. High salary offers go for manager approval
7. Approval/Rejection notification sent
8. Candidate hired
